# Задание 3

## Задача 1

### Текст задачи
Опишите функции, осуществляющие основные математические операции над комплексными числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать готовый класс Complex запрещено. ### Алгоритм решения
Запросить у пользователя действительную и мнимую(часть с I) часть 2 комплексных чисел
Далее с помощью собственно созданного типа Complex хранящего коэффициенты пользователя
Вызвать функции передав туда коэффициенты комплексных чисел
По правилам математики для
Сложение: складывается действительная часть с действительной мнимая с мнимой
Вычитание: тоже самое как у сложения только вычитание
Умножение: формула z = (a1a2 – b1b2) + i(a1b2 + a2b1)
Деление: Ищем сопряжённое число это а2*а2+b2*b2 и делим на него по формуле

Возведение в степень: Проверка степени-n на корректность (n>-1)
При n=0 выводится 1 
При n=1 выводится  исходное число 
При n>1 возводим число С в степень n/2(целочисленное деление )через POW
Получаем число С1 умножаем его само на себя 
Получаем число С2  
далее проверка если изначальное n четное то это нужное нам число 
Если n не четное то умножаем С2 на С и получим нужное число (эта часть из за целочисленного деления)
